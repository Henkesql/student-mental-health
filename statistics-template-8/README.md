# Studenters psykiska hälsa – analys

Detta projekt analyserar hur sömn och akademisk press påverkar studenters depressionsnivå.

## 📊 Funktioner
- SQLite-databas med studentdata
- SQL-analyser (GROUP BY, AVG, CASE)
- Visualisering med Google Charts
- Diagram: Sömn vs depression
- Diagram: Akademisk press vs depression

## 🛠️ Tekniker
- SQLite
- SQL
- JavaScript
- Google Charts
- HTML/CSS

## 📈 Resultat
- Studenter som sover mindre än 5 timmar har högst depressionsnivå
- Akademisk press verkar ha större påverkan än sömn
- Det är inte bara mängden arbete som spelar roll, utan hur pressen upplevs

1. Öppna `index.html`
2. Diagrammen visas i webbläsaren
